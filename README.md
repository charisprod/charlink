### Getting Started
- [Fork this repository](https://github.com/charisprod/charlink/fork).
- Or run the following command to create a new project with this template:

```bash
npx create-next-app@latest app --example https://github.com/charisprod/charlink
```

---

### Start Development
> Use any editor to work on editing. See: [App To Use](https://charlink-docs.charisprod.xyz/get-started/app-to-use).

- First, run the development server:

```bash
npm run dev
```

- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Preview
> To avoid failures due to errors or typo, it would be nice to do this step first before deploying to your website.

- Build your development.
```bash
npm run build
```

- If there are no issues, everything is ready to deploy.

- Or run the build preview using the following command:

```bash
npm start
```
